# **简介**
> 
## **性质**
**东南学苑**，英文“**Southeast Study Union**”，格言是“追求卓越，唯我独尊”，是于2018年3月成立于福建福州的青年学生组织，以助我等自主、团结、钻研、拼搏，做实事求是解放青年派。
